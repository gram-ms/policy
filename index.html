Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo **n8n** coleta, usa, armazena e compartilha informações obtidas através das APIs da Meta (Facebook, Instagram) e das integrações configuradas pelo usuário no n8n.

1. Dados coletados

Coletamos apenas os dados estritamente necessários para a execução das automações configuradas pelo usuário (por exemplo: IDs de páginas, tokens de acesso, conteúdos públicos de páginas ou contas conectadas quando autorizado).

2. Uso dos dados

Os dados são usados exclusivamente para executar e gerenciar fluxos/integrações do usuário no n8n, publicar conteúdos, ler métricas ou responder webhooks conforme as permissões concedidas.

3. Compartilhamento

Não vendemos nem compartilhamos dados com terceiros para fins publicitários. Dados podem ser enviados a serviços terceiros somente quando configurado explicitamente pelo usuário (ex.: destino de arquivos).

4. Armazenamento e retenção

Os dados são armazenados apenas conforme necessário para a funcionalidade do usuário. O usuário é responsável pela retenção e exclusão dos dados dentro do seu ambiente n8n.

5. Segurança

Recomendamos que o usuário proteja seu ambiente n8n (TLS, backups, controle de acesso). O aplicativo não tem acesso a credenciais de terceiros exceto as fornecidas explicitamente para integração.

6. Permissões solicitadas

As permissões solicitadas (ex.: business_management, pages_read_engagement, instagram_basic) são usadas apenas para as operações descritas acima.
